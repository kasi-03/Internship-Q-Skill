# Python Internship Projects

## Overview

This repository contains three Python projects completed as part of an internship program. The projects demonstrate practical applications of data analysis, data visualization, machine learning, and numerical computing using popular Python libraries such as Pandas, NumPy, Matplotlib, and Scikit-Learn.

---

## Project 1: Data Analysis and Visualization

### Objective

Load a CSV dataset, perform basic statistical analysis, and create visualizations to gain insights from the data.

### Technologies Used

* Python
* Pandas
* Matplotlib
* NumPy

### Features

* Load and process CSV files
* Calculate statistical measures such as mean, median, and summary statistics
* Generate visualizations:

  * Bar Charts
  * Scatter Plots
  * Correlation Heatmaps
* Identify trends and relationships within the dataset

### Sample Analysis

* Average value calculation for selected columns
* Correlation analysis between numerical features
* Visual representation of data distributions

---

## Project 2: House Price Prediction using Linear Regression

### Objective

Develop a machine learning model to predict house prices based on property features such as area, number of bedrooms, bathrooms, and furnishing status.

### Technologies Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib

### Features

* Data preprocessing and cleaning
* Handling categorical variables
* Train-test data splitting
* Linear Regression model training
* Performance evaluation using:

  * Mean Absolute Error (MAE)
  * R² Score
* Visualization of prediction results

### Model Workflow

1. Load housing dataset
2. Preprocess data
3. Train Linear Regression model
4. Evaluate performance
5. Predict house prices
6. Visualize actual vs predicted prices

### Sample Results

* Mean Absolute Error (MAE)
* R² Score indicating model accuracy
* Graphical analysis of prediction performance

### Technologies Used

### Features

* Matrix Subtraction
* Matrix Multiplication
* Matrix Transpose

### User Interface

The application allows users to:

1. Enter matrix dimensions
